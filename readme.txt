/* Create a new repository on GitHub.*/

	* Open Git Bash.
	* git init
	* git add .
	* git commit -m "First commit"
	* git remote add origin remote repository URL
	* git remote -v
	* git push origin master
	* If error: 
		* git push **-u** origin master
	* or 
		* git push -f origin master
